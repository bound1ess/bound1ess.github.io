Installation

Run in the terminal (in your project's root directory): 

composer require --dev bound1ess/essence

Congrats, you're all set!

Basics

In the examples below I will use both expect and should assertion styles.

I will show you how to use every single matcher that comes with Essence. You can also create your own matchers, more about it in the advanced section of this tutorial.

Let's get started. First, we'll take a look at EqualMatcher: 

this("string")->should_be_equal_to("foobar")->go();

This line will throw an exception (Essence\Exceptions\AssertionException), because "string" is obviously not equal to "foobar". Let's do one more: 

expect(this(123))->toBeEqual(123)->go();

This will (just as you might expect) return true. But what about this one? 

this(new stdClass)->should->be->equal(new stdClass)->go();

This example will fail. Can you guess why? That's because Essence always performs strict (===) checks. I hope it's all clear to you, so let's continue on.

Now let's review TrueMatcher, FalseMatcher, PositiveMatcher, EmptyMatcher, NullMatcher: 

expect(this(false))->to->be->true()->go();
it(false)->shouldBeFalse->go();
expect(this(0))->to_be_ok()->go();
this([])->should_be_empty->go();
expect(this(null))->not_to_be_null->go();
